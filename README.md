# Image Segmentation Program
Small **Python** program performing image segmentation using **k-means clustering algorithm**

### Supported Image Types
- **JPEG(.jpg)**
- **PNG(.png)**

### External Python Libraries
- **Scikit-Learn & Scikit-Image**
  - Provides basics of k-means algorithm and image image loading.
- **Numpy**
  - Provides matrix operations
- **Pillow(PIL)**
  - Support image convert
- **Tkinter**
  - Graphing GUI
